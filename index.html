Add your API key and change your password

1Add your Anthropic API key

Get one at console.anthropic.com → API Keys → Create Key. Stored locally in your browser only — never shared.

2Change password

Default password is your university student no — change it to something personal.

3Backing up your data

Your data lives in this browser on this device. To keep it safe, export a CSV every Friday — ZulFolio will remind you automatically.

  1. Go to Export in the sidebar → All entries — CSV
  2. Save the file somewhere safe — Google Drive, OneDrive, or your desktop

⚠️ If you clear your browser data or switch browsers, your entries will be lost without a backup.
